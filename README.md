# go-htmx-playground

This is a Go + HTMX demo project.

See [techformist.com/go-htmx-book-tracker](https://techformist.com/go-htmx-book-tracker) for more details.

## Get Started

1. Install Go on your computer
2. Install `nodemon`
3. Clone this repo
4. Run npm install

Start the server -
```bash
nodemon --exec go run main.go
```
